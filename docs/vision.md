The goal is to create a way to track ideas / strategies / products in a queriable way, allowing people to brouse and arrive at the solutions from any approach. 
There should also be scripts that keep highlighting items to add / update, and a human should review/promote the changes.

tldr: A searchable database of assistive technology strategies, barriers, access needs, and fit rules.

Version 0.1 will support:
- Solutions
- Dimensions / constraints
- Fit rules
- Sources
- Search terms
- SQL-based editing
- Read-only public browsing later
- update scripts later
